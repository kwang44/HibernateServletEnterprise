Welcome to the WangWickWilliam Reimbursement API

RESTful endpoints and supported methods(all consume and produce in Json):


/login

resource = login tokens(implemented via HttpSession)

POST:to attempt login/create a session {"username":String,"password": String}

/users

resource = users

POST:to create an account {"username":"","password":"","firstname":"","lastname":"","email":""}

GET:@noParam to get all users from persistence @{optionalGetParam = username} to get single user by username

/users/reimbursements

resource = users

POST:to create a reimbursement request { "amount": 00.00, "description":"", "type_id": 0 == dining 1 == travel 2 == misc }

GET:@noParam to get all reimbursements from persistence @{optionalGetParam = id} to get single user's reimbursements

PUT:resolves 0 to n reimbursements [{"id":1, "approved":false},...,{"id":n, "approved":true} ]
